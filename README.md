# Event Monitoring Dashboard

Веб-приложение для мониторинга системных событий с фильтрацией и виртуализацией списка.

## Быстрый старт

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

Приложение откроется по адресу `http://localhost:5173`

### Сборка для продакшена
```bash
npm run build
```

### Preview продакшен сборки
```bash
npm run preview
```


#### Технические решения

**Используется `HashRouter` для совместимости с GitHub Pages
**Система фильтрации**: поиск по тексту, выбор приложения, диапазон дат
**Персистентность фильтров**: состояние сохраняется в URL и восстанавливается при перезагрузке
